## âš¡ Quick Start

### 1. Clone the Repo
```bash
git clone <repo-url>
cd finzen project
```

### 2. Backend Setup
```bash
cd backend
npm install
# Create .env file:
# MONGO_URI=your_mongodb_connection_string
# PORT=5000
npm run dev
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
# Create .env file:
# VITE_API_BASE=https://your-backend-url.onrender.com/api
npm run dev
```
